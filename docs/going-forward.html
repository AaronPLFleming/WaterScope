End User Perspective (Saj)

End User Workflow

The current workflow employed by WaterScope in Tanzania (i.e. using a sterile syringe to manually push 100ml of water through the filter) can be summarised in the flow diagram below.

If our prototype was incorporated by any end user, the workflow is largely the same, with the changes outlined in red and the advantages and disadvantages explained below.

Some of the advantages and disadvantages of implementing our prototype product into the WaterScope workflow include:

Advantages

  • Multiple samples can be filtered in one go either on site (especially if there are lots of different water samples in close proximity) or in the lab.
  • Less effort is required to filter once the vacuum is set up - i.e. no force is required to push the syringe.
  • Once the vacuum filtration apparatus is set up it only needs intermittent monitoring, so the user can concentrate on other tasks e.g. extracting water samples from other sources close by.

Disadvantages

  • Vacuum apparatus must be set up at each site - this may be time consuming and inconvenient (hopefully made easier by our assembly instructions).
  • The vacuum apparatus must be taken with the rest of the equipment for the day. This may be a problem as it is somewhat bulky and fragile, so the end user must be careful for it not to get damaged. Furthermore, a large proportion of the equipment is unlikely to be readily available at a reasonable price in country.

User Assistance Documentation
